Flight.FlightLib.Dynamics.FrameTransform — Type

Specify a reference frame $c$ with respect to another $b$.

$c$ is defined by:

  • $r_{bc}^{b}$: Position vector from $b$'s origin $O_b$ to $c$'s origin $O_c$, projected on $b$'s axes $Ɛ_b$
  • $q^b_c$: Rotation quaternion from $b$'s axes $Ɛ_b$ to $c$'s axes $Ɛ_c$

FrameTransform objects are callable; the syntax t_bc(x_c) is equivalent to translate(t_bc, x_c).

Flight.FlightLib.Dynamics.translate — Method
translate(t_bc::FrameTransform, r_cP_c::AbstractVector{<:Real})

Translate a 3D position vector from one reference frame to another.

Given the FrameTransform t_bc from frame $b$ to frame $c$ and the position vector of some point $P$ in $c$ ($r_{cP}^{c}$), return the position vector of $P$ in $b$ ($r_{bP}^{b}$).
